MacGuffin additions agreed so far
=================================

Set rule
--------
- Only create a named set when it has at least 3 macguffins.
- Otherwise, keep the macguffin in General.
- Set labels should stay short for embeds.
- Achievement titles should be one word when possible, two words max.

New set candidates
==================

Flavor note
-----------
- Flavor text should match the tone of the source movie/show.
- Lynch should feel spare, dreamlike, and uneasy.
- Cronenberg should feel clinical, bodily, and calm in a wrong way.
- Tarantino should feel casual, quotable, and swaggering.
- Coen should feel dry, fatalistic, and plainspoken.
- Carpenter should feel blunt, pulpy, and midnight-movie tough.
