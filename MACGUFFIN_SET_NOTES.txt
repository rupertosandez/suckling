MacGuffin additions
===================

Set rules
---------
- Only create a named set when it has at least 3 macguffins.
- Otherwise, keep the macguffin in General.
- Set labels should stay short for embeds.
- Achievement titles should be one word when possible, two words max.
- Flavor text should match the tone of the source movie/show.
